# Links Manager - Chrome Extension

Chrome расширение для удобного управления короткими ссылками.

## Установка

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode)
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку `extension`

## Использование

1. Нажмите на иконку расширения в панели инструментов Chrome
2. В открывшемся окне вы можете:
   - Просматривать все ссылки
   - Фильтровать по группам (dev, prod, personal)
   - Искать ссылки по названию, URL или описанию
   - Создавать новые ссылки
   - Редактировать существующие ссылки
   - Удалять ссылки
   - Открывать ссылки через короткий URL

## Настройки

В нижней части окна расширения можно настроить URL API сервера (по умолчанию: `http://localhost:4000`).

## Требования

- Phoenix API сервер должен быть запущен на `http://localhost:4000` (или другом URL, указанном в настройках)
- API должен быть доступен без аутентификации (или с настроенной аутентификацией)

## Разработка

Для разработки расширения:

1. Внесите изменения в файлы расширения
2. В `chrome://extensions/` нажмите кнопку обновления (↻) для расширения
3. Откройте popup и проверьте изменения

## Структура файлов

- `manifest.json` - манифест расширения
- `popup.html` - HTML интерфейса
- `popup.js` - логика работы интерфейса
- `styles.css` - стили интерфейса
- `background.js` - фоновый скрипт
- `content.js` - контентный скрипт для работы на страницах

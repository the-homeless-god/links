# Начало работы

В этом разделе вы найдете всю необходимую информацию для начала работы с Links API.

## Что вам нужно знать

Links API — это система управления ссылками, состоящая из:

- **Backend API** на Elixir/Phoenix
- **Chrome Extension** для удобного доступа
- **Keycloak** для авторизации (опционально)

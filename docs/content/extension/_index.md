# Chrome Extension

Chrome расширение для Links API предоставляет удобный интерфейс для управления ссылками прямо из браузера.

## Установка

### Из исходников

1. Клонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/links.git
cd links/extension
```

2. Установите зависимости:
```bash
npm install
```

3. Соберите расширение:
```bash
npm run build
```

4. Загрузите в Chrome:
   - Откройте `chrome://extensions/`
   - Включите "Режим разработчика"
   - Нажмите "Загрузить распакованное расширение"
   - Выберите папку `extension/dist`

### Из релиза

1. Скачайте последний релиз с GitHub
2. Распакуйте архив
3. Загрузите папку `dist` как расширение в Chrome

## Настройка

После установки настройте URL API в настройках расширения:

1. Откройте popup расширения
2. Перейдите в настройки
3. Укажите URL вашего API (по умолчанию: `http://localhost:4000`)

## Использование

### Добавление ссылки

1. Откройте popup расширения
2. Нажмите "Добавить ссылку"
3. Заполните форму:
   - Название
   - URL
   - Описание (опционально)
   - Группа (опционально)
4. Нажмите "Сохранить"

### Просмотр ссылок

Все ваши ссылки отображаются в popup расширения. Вы можете:

- Фильтровать по группам
- Искать по названию
- Открывать ссылки в новой вкладке
- Редактировать и удалять ссылки

### Авторизация

Расширение поддерживает:

- **Guest режим** — автоматически использует гостевой токен
- **Keycloak** — авторизация через Keycloak (требует настройки)

# Публичные ссылки

Публичные ссылки доступны всем пользователям без авторизации по пути `/u/:name`.

## Создание публичной ссылки

Установите `public: true` при создании ссылки:

```bash
curl -X POST http://localhost:4000/api/links \
  -H "Content-Type: application/json" \
  -H "X-Guest-Token: guest" \
  -d '{
    "name": "public-link",
    "url": "https://example.com",
    "public": true
  }'
```

## Доступ к публичной ссылке

После создания ссылки с именем `public-link`, она будет доступна по адресу:

```
http://localhost:4000/u/public-link
```

Этот URL работает **без авторизации** и автоматически перенаправляет на указанный URL.

## Использование

Публичные ссылки полезны для:

- Обмена ссылками с пользователями без аккаунта
- Создания коротких ссылок для публикации
- Интеграции с внешними системами

## Ограничения

- Имя ссылки должно быть уникальным среди всех публичных ссылок
- Публичные ссылки видны всем пользователям в списке ссылок

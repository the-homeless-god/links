# Настройка GitHub Pages для документации

## Первоначальная настройка

1. Перейдите в Settings → Pages вашего репозитория на GitHub
2. В разделе "Source" выберите "GitHub Actions"
3. Сохраните настройки

## Обновление baseURL

После настройки GitHub Pages обновите `baseURL` в `docs/config.toml`:

```toml
baseURL = 'https://YOUR_USERNAME.github.io/links/'
```

Замените `YOUR_USERNAME` на ваш GitHub username.

Если ваш репозиторий называется по-другому, измените путь соответственно.

## Автоматическая публикация

После настройки документация будет автоматически публиковаться при каждом пуше в ветку `main` или `master`, если изменены файлы в директории `docs/`.

## Ручной запуск

Вы можете запустить публикацию вручную:

1. Перейдите в Actions → "Deploy Documentation"
2. Нажмите "Run workflow"
3. Выберите ветку и запустите

## Локальная проверка

Перед коммитом проверьте документацию локально:

```bash
cd docs
hugo server -D
```

Откройте `http://localhost:1313` в браузере.

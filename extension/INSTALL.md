# Инструкция по установке Chrome Extension

## Шаги установки

1. **Откройте Chrome и перейдите в настройки расширений:**
   - В адресной строке введите: `chrome://extensions/`
   - Или: Меню → Дополнительные инструменты → Расширения

2. **Включите режим разработчика:**
   - В правом верхнем углу страницы включите переключатель "Режим разработчика" (Developer mode)

3. **Загрузите расширение:**
   - Нажмите кнопку "Загрузить распакованное расширение" (Load unpacked)
   - Выберите папку `extension` из проекта

4. **Проверьте установку:**
   - В списке расширений должно появиться "Links Manager"
   - В панели инструментов Chrome должна появиться иконка расширения

## Настройка

1. **Запустите Phoenix API сервер:**
   ```bash
   cd /path/to/elixir_backend
   mix phx.server
   ```

2. **Настройте API URL в расширении:**
   - Откройте расширение (кликните на иконку)
   - В нижней части окна введите URL API (по умолчанию: `http://localhost:4000`)
   - Нажмите "Сохранить"

## Использование

### Основные функции

- **Просмотр ссылок:** Откройте расширение, чтобы увидеть все ваши ссылки
- **Фильтрация:** Используйте вкладки для фильтрации по группам (dev, prod, personal)
- **Поиск:** Введите текст в поле поиска для поиска по названию, URL или описанию
- **Создание ссылки:** Нажмите "+ Новая ссылка" и заполните форму
- **Редактирование:** Нажмите "Редактировать" на любой ссылке
- **Удаление:** Нажмите "Удалить" на любой ссылке
- **Открытие:** Нажмите "Открыть" для перехода по короткой ссылке

### Быстрое создание ссылки

1. **Из текущей страницы:**
   - Откройте расширение
   - Нажмите "+ Новая ссылка"
   - URL текущей страницы будет автоматически заполнен

2. **Через контекстное меню:**
   - Правый клик на любой странице
   - Выберите "Создать короткую ссылку"
   - Откроется расширение с предзаполненными данными

## Устранение неполадок

### Расширение не загружается
- Убедитесь, что включен "Режим разработчика"
- Проверьте, что вы выбрали правильную папку `extension`

### Ошибка при загрузке ссылок
- Проверьте, что Phoenix API сервер запущен
- Проверьте URL API в настройках расширения
- Убедитесь, что CORS настроен правильно в Phoenix

### Ссылки не создаются
- Проверьте, что имя ссылки уникально
- Убедитесь, что URL валидный (начинается с http:// или https://)
- Проверьте консоль браузера на наличие ошибок
